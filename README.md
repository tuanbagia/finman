## 🌟 What is FinMan anyway?
It's a next-gen, buttery smooth financial tracking system built for those who want their dashboards looking as clean as their fit. Track your allowance, side hustles, coffee addictions, and late-night impulse buys in real-time. With stunning interactive charts, smooth micro-animations, and full mobile responsiveness, checking your bank balance doesn't have to be a horror movie anymore. 💀

### ✨ Features (The Flex List)
- 🤖 **AI Financial Sidekick**: Built-in chat assistant powered by **Google Gemini (Gemini 2.5 Flash)** that actually analyzes your transactions, gives personalized savings advice, and tells you to stop ordering takeout. 💅✨
- 📊 **Visuals on Point**: Beautiful, interactive charts powered by Recharts so you can watch where your money is running off to.
- 🪄 **Buttery Animations**: Smooth transitions powered by Framer Motion (it's satisfying fr).
- ⚡ **Light Speed**: Powered by Vite + React 18. Zero lag, all speed.
- 📱 **Responsive Layout**: Looks fire on your desktop, tablet, or phone.
- 🔒 **Cloud Sync**: Connected to Firebase to keep your funds tracked and synced everywhere securely.
- 🌙 **Dark Mode Support**: Because light mode at 2 AM is a crime. 🦇

## 🚀 Spin It Up Locally (Time to cook 🧪)

Want to run this masterpiece on your own machine? Let's get you set up in seconds.

### 1️⃣ Clone the Repo
```bash
git clone https://github.com/tuanbagia/finman.git
cd finman
```

### 2️⃣ Install Dependencies
We recommend `pnpm` because it's fast, but `npm` or `yarn` works too!
```bash
pnpm install
# or
npm install
```

### 3️⃣ Add the Secret Sauce (Env Variables)
Create a `.env` file in the root folder and paste your Firebase and Gemini configs:
```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_project.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_messaging_sender_id
VITE_FIREBASE_APP_ID=your_app_id
VITE_GEMINI_API_KEY=your_gemini_api_key
```

### 4️⃣ Run the Dev Server
```bash
pnpm dev
# or
npm run dev
```

Now open **`http://localhost:5173`** in your browser and start tracking like a pro! 🚀
